Un proyecto simple para analizar las diferencias en la creación de pruebas usando el método tradicional vs utilizando IA generativa para crear los casos de prueba.

# Ejecutar las pruebas

Asumiendo que Python está instalado, desde una terminal:

1. Clonar el repositorio

```bash
git clone https://github.com/andressero/ShoppingCartTesting.git
```

2. Entrar al repositorio:
```bash
cd ShoppingCartTesting
```

3. Si es necesario, crear un entorno virtual de Python:

```bash
python3 -m venv .env
```

4. Activar el entorno virtual:

```bash
source .venv/bin/activate
```

5. Instalar pytest:
```bash
pip install -U pytest
```

6. Ejecutar pytest:
```bash
pytest
```
